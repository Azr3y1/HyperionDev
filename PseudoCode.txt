Pseudocode is a way for programmers to visualize their code in an easy to read format before actually writing any code. An important step when beginning any project as it outlines work/steps that must be taken in the code writing process. Writing pseudocode will help the programmer implement algorithms into their code ensuring a step-by-step process is followed in the workflow.
